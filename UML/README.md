## Adding New Diagrams

When adding new UML diagrams:

1. **Use descriptive filenames**: Use kebab-case (e.g., `user-authentication-sequence.png`)
2. **Supported formats**: `.jpg`, `.png`, `.svg`, `.pdf`
3. **Update this README**: Add a brief description of what the diagram shows
4. **Version control**: Commit diagrams to the repository for team access
5. **Keep diagrams updated**: Update diagrams when system architecture changes

## Tools Recommended

- **Draw.io / diagrams.net**: Free, web-based UML tool
- **PlantUML**: Text-based UML diagram creation
- **Lucidchart**: Professional diagramming tool
- **StarUML**: Desktop UML modeling application
